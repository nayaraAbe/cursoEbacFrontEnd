Formulários

São elementos que permitem o usuário de mandar dados ao servidor web, isso facilita a coleta de dados, como em cadastros, pesquisas, feedback e compras online.

"form"

É a tag principal para se criar formulários, ela encapsula todos os outros elementos.

"input"

Campo utilizado para entrada de dados, podendo ser qualquer informação vinda do usuário, como senha, pesquisa, e-mail entre outros.

Dentro da tag "input" pode-se utilizar "placeholder" que nomeia com o que o campo de texto deve ser preenchido.

"label" indica fora da caixa de texto o que o usuário deve preencher, também possui o comportamento de indicar qual campo deverá ser preenchido com o click do mouse

type=""

Utilizando o tipo de texto para celular/telefone "type="tel"" o tetxo se torna responsivo, o que facilita para usuários de outros tipos de aparelhos como o celular e tablet, tendo somente o teclado numerico

"type="number"" é usando para definir uma caixa numerica podendo ter os atributos de "max" para definir o número maximo que se possa colocar e também "min" para o número minimo. Pode-se indicar o salto entre um número e outro com "step".

"type="file"" cria uma caixa para de mandar arquivo

"type="color""cria uma caixa de celeção de cores

"type="checkbox"" usado como caixa de marcação

Campo de alternativa

Utiliza o "type="radio"" para criar opções/alternativas de respostas para o usuario


Qual é sua cor favorita?

Caixa de opções(combo box)

Utiliza as tags "select" e "option para estar criando a caixa de opções


"textarea"

Se ultiliza a tag "textarea" para criar um campo de texto maior, normalmente utilizado para o usuario deixar feedbacks e comentários

"button"

a tag "button" é utilizada para criar botões, podendo ser usado com atributo

"type="password""

Esse tipo de atributo também é responsivo, pois garante a seguraça de senhas do usuário
