Adicionando tipos de midias

Imagem

Para de adicionar uma imagem se utiliza a tag "img" juntamente com o atributo "src" que inseri um recurso externo.
E podemos usar o atributo "alt"para ter uma alternativa de leitura da imagem.



Audio

Para adicionar audio se utiliza a tag "audio" juntamente com o atributo "src" para inserir um recuso externo.
E também se utiliza o atribut "Controls para inserir um modo de visualização de um "play"


Video

Para Adicionar video se utiliza a tag "video"juntamente com o atributo "src" para inserir um recurso externo.
E como na tag "audio" em "video" também se utiliza o atributo "controls"para se dar "play" no conteudo adicionado.


Midias externas

Também utilizamos midias externas capiturando o link e os atribuindo conforme seu formato.
