Antigamente os containers não possuiam nomenclaturas sendo classificados somente como "div" o que dificultava nas suas criações, portanto foi facilidado:

Tag "header" é um container normalmente onde se possui o conteudo de introdução ou links de navegação importantes.

Tag "nav" usada para conjunto de links de navegação da página dentro da tag "header".

main:

O que a tag "main" faz?

A tag "main" é um container onde o conteudo principal da pagina se encontra podendo ser artigos, postagens, seções de texto, formulários, imagens/mídia, tabelas, etc.

section:

O que a tag "section" faz?

A tag "section" Utilizada para fazer seções dentro do corpo principal "main".

article:

O que a tag "article" faz?

A tag "article" é utilizada para criar artigos determinando o conteudo dentro das seções, aconpanhadas normalmente com um titulo.

div:

O que a tag "div" faz?

A tag "div" é um container ainda funcional resultante do funcionamento com o CSS e JavaScript.

aside:

O que a tag "aside" faz?

A tag "aside" é um container utilizado na lateral da pagina seu uso é semantico, pois faz a utilização do CSS.

CSS:

O que é CSS?

CSS é uma linguagem de estilização, usada no front-end para dar estilo a página.

Tag "footer" é um container geramnete utilizado para agrupar informações que encerram um conteúdo.
